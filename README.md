# bybit_card_tracker

A new Flutter project.
A Flutter application designed to track and manage Bybit Card transactions and balances.

## Features

- **Balance Monitoring:** Real-time tracking of card-funding account balances.
- **Transaction History:** Categorized view of recent card expenditures.
- **Analytics:** Visual charts showing spending trends.
- **Security:** Secure storage for Bybit API credentials.

## Getting Started

1. Clone the repository.
2. Run `flutter pub get`.
3. Create a `.env` file for your API configuration.
4. Run `flutter run`.

## Architecture

This project follows Clean Architecture principles to ensure scalability and testability.
